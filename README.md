## GitHub Pages setup

1. Create a GitHub repository, for example `sustenesis` or `sustenesis-site`.
2. Upload all files in this folder to the repository root.
3. Go to **Settings ‚Üí Pages**.
4. Choose **Deploy from a branch**.
5. Select the `main` branch and the root folder `/`.
6. Set the custom domain to `sustenesis.com`.
7. In your DNS provider, point `sustenesis.com` to GitHub Pages.

## DNS notes

For an apex domain such as `sustenesis.com`, GitHub Pages commonly uses A records pointing to GitHub Pages IP addresses and a CNAME record for `www`. Check GitHub's current documentation before editing DNS records.
